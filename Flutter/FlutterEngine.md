## Engine安装

### 工具：

1、下载Chromium提供的部署工具[depo_tools](https://chromium.googlesource.com/chromium/tools/depot_tools/)

```
git clone https://chromium.googlesource.com/chromium/tools/depot_tools
```

2、配置depo_tools的环境变量

```
open ~/.bash_profile

export PATH=/opt/depot_tool/:$PATH
```

3、安装ant编译工具

```
brew install ant
```
